# Town Hall — Ground Rules

**Purpose:** Make decisions faster, not slower. The town hall exists to give the solo founder (Scott) multi-perspective analysis in minutes, not hours.

---

## Ground Rules (every discussion)

1. **Start with the user problem** — What remote worker pain point are we solving?
2. **15-minute max per topic** — Hard stop. Unresolved = escalate to founder.
3. **Evidence over opinion** — Cite user feedback, data, or technical constraints.
4. **Local-first constraint check** — Does this break offline-first? 30-second sanity test.
5. **First principles** — Strip away assumptions. What's actually true? What's just convention?
6. **No hedge-everything answers** — Pick a side. If you see tradeoffs, spell them out.

## Decision Framework

**Stop debating when ANY trigger fires:**
- 15 minutes elapsed
- 3+ back-and-forth cycles on same point
- Discussion shifts from "how" to "whether" (scope creep signal)

**Commitment protocol:**
1. **Founder breaks all ties** — Models present 2-sentence condensed positions
2. **24-hour implementation rule** — Decisions get prototyped within 24 hours
3. **Pilot and iterate** — Favor reversible decisions over perfect ones
4. **Revert threshold** — Decision stands unless testing shows >30% negative impact

## Weighting Disagreements

**By domain:**
- Technical architecture → Claude (strongest systems reasoning)
- Long context / docs → Gemini (1M context window)
- Logic verification → DeepSeek / o3 (reasoning chains)
- Broad alternatives → GPT-4o (breadth of knowledge)
- Business priority → **Founder always wins**

**By split:**
- 3-1 or better: Go with majority, log dissent
- 2-2: Founder decides immediately
- Unanimous against founder: 48-hour cooling period, then founder decides

## Visibility

- **Every discussion saves to `cowork-brain/discussions/` as a `.md` file**
- Slack #ai-comms gets a one-line notification pointing to the doc — NOT the full discussion
- Anyone with repo access can read and edit — no Slack account needed
- Discussions are named `YYYY-MM-DD-topic.md` with decision checkboxes at the bottom
- The town hall is not a black box — it's a meeting transcript anyone can edit

## Anti-Spiral Rules

- No model may respond to its own previous point (prevents echo chamber)
- Synthesis round is mandatory when 2+ models participate
- If total cost exceeds $0.10 for a single discussion, auto-stop
- Maximum 3 rounds of debate before forced synthesis
